Bienvenue sur Songo

Le jeu de mancala camerounais, jouable en solo, en local ou en ligne — directement dans votre navigateur.

🌍 Patrimoine camerounais 🤖 IA intégrée 🌐 Multijoueur en ligne 📱 Responsive
1

Qu'est-ce que le Songo ?

Le Songo est un jeu de société traditionnel camerounais appartenant à la grande famille des Mancalas — l'un des plus vieux types de jeux au monde. Il se joue sur un plateau rectangulaire comportant 12 fosses et 2 greniers. L'objectif est de capturer plus de graines que l'adversaire.

Cette application web vous permet d'y jouer sans aucune installation, depuis n'importe quel navigateur moderne. Elle a été développée dans le cadre du cours INF 222 — Technologies Web à l'Université de Yaoundé I.

🎮

Aucune installation

Le jeu s'ouvre directement dans votre navigateur, sans plugin ni téléchargement.

🏆

3 modes de jeu

Affrontez l'IA, un ami sur le même écran, ou des joueurs distants en ligne.

🔊

Expérience immersive

Animations de graines et effets sonores pour chaque action de jeu.

3

Lancer votre première partie

1
Ouvrir l'application

Ouvrez le fichier index.html dans votre navigateur (Chrome, Firefox, Edge). Vous arrivez sur le menu principal.

2
Choisir un mode de jeu

Cliquez sur l'un des trois boutons du menu : Solo contre l'IA, 2 Joueurs ou En ligne.

3
Sélectionner la difficulté (mode Solo)

En mode Solo, choisissez entre Facile, Moyen ou Difficile selon votre niveau d'expérience avec le Songo.

4
Jouer votre tour

Cliquez sur l'une de vos fosses (celles du bas) pour semer les graines. Le jeu gère automatiquement les captures et les tours bonus.

5
Fin de partie

La partie se termine quand un joueur ne peut plus jouer. Le joueur ayant le plus de graines dans son grenier remporte la partie.

💡

Astuce : Le bouton Règles dans l'interface de jeu charge le détail des règles à tout moment, sans quitter la partie en cours.

5

Règles du jeu

🎯

Objectif : Capturer le plus grand nombre de graines dans votre grenier. La partie se termine quand un joueur n'a plus de graines à semer.

①
Semer les graines

À votre tour, cliquez sur l'une de vos fosses (non vide). Toutes les graines de cette fosse sont prélevées et semées une par une dans les fosses suivantes, dans le sens antihoraire.

②
Le grenier adverse est sauté

Lors du semis, le grenier de l'adversaire est ignoré : aucune graine n'y est déposée. Votre propre grenier, lui, reçoit bien une graine au passage.

③
Tour bonus

Si la dernière graine de votre semis atterrit dans votre grenier, vous obtenez un tour bonus et pouvez rejouer immédiatement.

④
Capture

Si la dernière graine tombe dans une fosse vide de votre camp, et que la fosse en face (côté adverse) contient des graines, vous capturez toutes ces graines + la vôtre et les placez dans votre grenier.

⑤
Fin de partie

Quand un joueur n'a plus de graines à jouer, toutes les graines restantes dans le camp adverse vont dans le grenier de l'adversaire. Le joueur avec le plus de graines gagne.

⚠️

Attention : Vous ne pouvez jouer que les fosses de votre propre camp. Cliquer sur une fosse adverse ou une fosse vide n'a aucun effet.

6

Intelligence artificielle

En mode Solo, l'ordinateur joue selon l'un des trois niveaux de difficulté choisi au démarrage. L'IA analyse le plateau et sélectionne son coup automatiquement.

😊 Facile

L'IA joue de manière aléatoire. Idéal pour découvrir le jeu ou jouer avec des enfants.

🤔 Moyen

L'IA privilégie les captures et les tours bonus. Un vrai défi pour les joueurs intermédiaires.

😤 Difficile

L'IA utilise un algorithme heuristique pour maximiser ses graines. Réservé aux joueurs expérimentés.

🧠

Conseil : Commencez par le niveau Facile pour maîtriser les règles, puis progressez vers Moyen. Le niveau Difficile simule un joueur humain expérimenté.

8

Foire aux questions

Oui, les modes Solo et Local 2 joueurs fonctionnent entièrement hors ligne. Seul le mode En ligne nécessite une connexion active pour communiquer avec le serveur WebSocket. Notez que les règles (chargées via AJAX) nécessitent un serveur local si vous ouvrez le fichier directement.

En bas de l'interface de jeu, un bouton Nouvelle partie réinitialise immédiatement le plateau avec les mêmes paramètres (mode et difficulté). Le bouton Menu vous ramène à la sélection de mode.

Le chargement des règles utilise AJAX (requête HTTP). Si vous ouvrez index.html directement en double-cliquant (protocole file://), certains navigateurs bloquent cette requête. Solution : utilisez un serveur local (Live Server dans VS Code, ou python -m http.server) pour servir les fichiers.

Oui, l'interface est responsive et s'adapte aux petits écrans. Le plateau se réorganise verticalement et les fosses restent facilement cliquables au doigt. L'expérience optimale reste sur ordinateur pour le mode Local 2 joueurs.

En cas de déconnexion de l'adversaire, le serveur WebSocket ferme la salle. Vous serez redirigé vers le lobby avec un message d'information. Vous pouvez alors créer ou rejoindre une nouvelle salle.

Une capture se produit uniquement si : (1) votre dernière graine semée tombe dans une fosse vide de votre camp, ET (2) la fosse directement en face (côté adversaire) contient au moins une graine. Dans ce cas, vous prenez toutes les graines de la fosse adverse + la graine qui vient d'arriver, et vous les placez dans votre grenier.

Glossaire

TermeDéfinition
MancalaFamille de jeux de stratégie africains et asiatiques basés sur le semis et la capture de graines dans des fosses.
FosseAlvéole du plateau destinée à contenir des graines. Chaque joueur en possède 6.

SemisAction de distribuer les graines d'une fosse une par une dans les fosses suivantes.
CaptureAction de prendre les graines d'une fosse adverse dans certaines conditions.
Tour bonusTour supplémentaire accordé quand la dernière graine semée atterrit dans le grenier du joueur.
WebSocketProtocole de communication en temps réel permettant des échanges bidirectionnels entre navigateur et serveur.
AJAXTechnique JavaScript permettant de charger du contenu depuis un serveur sans recharger la page entière.
